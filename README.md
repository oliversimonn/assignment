# Building project 

```mvn clean install```

# Prerequisites

###Maven version 3.0+
###Java version 18+

# Running app

```java -jar assignment.jar --fooDivider=3 --faaDivider=5 --destination=CONSOLE|FILE```

If arguments are not present, default values are fooDivider 2, faaDivider 4 and destination is CONSOLE

```java -jar assignment.jar```
